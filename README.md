# Tic-Tac-Toe

## Overview
Tic-Tac-Toe is a simple two-player game where players take turns marking a space in a 3x3 grid. The first player to align three of their marks horizontally, vertically, or diagonally wins the game.

## Project Structure
```
tic-tac-toe
├── index.html        # Main HTML file for the game
├── css
│   └── style.css     # Styles for the game
├── js
│   └── script.js     # JavaScript logic for the game
├── tests
│   └── game.test.js  # Unit tests for the game logic
├── .gitignore        # Files and directories to ignore by Git
└── README.md         # Project documentation
```

## Getting Started
To run the Tic-Tac-Toe game, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd tic-tac-toe
   ```

3. **Open the `index.html` file** in your web browser to start playing.

## Features
- Two-player gameplay
- Visual indication of whose turn it is
- Restart button to reset the game
- Simple and responsive design

## Testing
Unit tests for the game logic can be found in the `tests/game.test.js` file. You can run these tests using a JavaScript testing framework of your choice.
